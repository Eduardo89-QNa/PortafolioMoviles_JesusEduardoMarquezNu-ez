# Proyecto 02 - Análisis de Datos con Dart

## Nombre del proyecto
Análisis de Datos con Dart (Aplicación de Consola)

## Objetivo del proyecto
Crear una aplicación de consola en Dart que lea datos desde un archivo JSON, permita buscar, filtrar, ver estadísticas y exportar los resultados.

## Problema que resuelve
Automatiza el análisis de datos almacenados en formato JSON, facilitando búsquedas, filtros y generación de reportes básicos.

## Tecnologías utilizadas
- Dart
- JSON (lectura y escritura)
- Programación Orientada a Objetos

## Conceptos aplicados
- Clases y objetos (`Registro`)
- Manejo de archivos JSON (`dart:io`, `dart:convert`)
- Null Safety
- Funciones de filtrado y búsqueda
- Menú interactivo en consola
